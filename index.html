Integer Sequence Dividing

10 Jan 2019

题目链接

http://codeforces.com/problemset/problem/1102/A

题意

将1-n的正整数分为两个集合，使得两个集合里面数字和的差值的绝对值最小，输出两个集合最小的绝对值差值。

分析

若1-n的和是偶数，则可以划分为两个和相等的集合，差值为0。若为奇数，两个集合差值为1。

code

//
// Created by fssqawj on 2019/1/10.
//

#include <bits/stdc++.h>

int main() {
    int n;
    scanf("%d", &n);
    printf((n * (n + 1)) & 3 ? "1\n" : "0\n");
    return 0;
}
